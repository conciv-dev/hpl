---
title: napl init
description: Create the .napl/ structure, lock.json, and an example prompt.
---

```text title="usage"
Usage: napl init

Options:
  -h, --help  Print help
```

Writes:

- `.napl/` — all derived state (`map.json`, `journal.jsonl`, `src/`, `ir/`, `attribution/`, `mapl/`),
- `.napl/lock.json` — pins the model (`claude-sonnet-5`) and backend (`claude-cli`),
- an example prompt,
- guard rails: `AGENTS.md` + `CLAUDE.md` in each generated tree, a `.claude/settings.json` denying edit tools on `.napl/src/**` (merge-updated), and a `pre-commit` hook running `napl status` (never overwrites an existing hook).

```bash
napl init
```
