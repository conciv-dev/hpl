---
title: napl blame
description: Git-blame-style line history for a generated file, replayed from the journal.
---

Every successful gen appends one line to `.napl/journal.jsonl` — gen number, timestamp, prompt diff, per-file patch. `blame` replays those patches to find, for each current line, the gen that last touched it.

```text title="usage"
Usage: napl blame [OPTIONS] [FILE]

Arguments:
  [FILE]  A generated file under .napl/src/<target>/

Options:
  -l, --line <LINE>  Blame only a single 1-based line
  -g, --gen <GEN>    Print the summary of a single gen journal entry
  -v, --verbose      Also show the prompt edit that caused each line
  -h, --help         Print help
```

```bash
napl blame .napl/src/react/greeting.tsx              # per line: the gen that last touched it
napl blame .napl/src/react/greeting.tsx --line 19 -v # one line, plus the prompt edit behind it
napl blame --gen 12                                  # one gen: module, prompt diff, files
```

Same algorithm class as `git blame`: untouched lines keep the oldest gen, lines shifted by insertions keep theirs, modified lines move to the editing gen.
