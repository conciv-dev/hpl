One native binary, `napl`. Prebuilt for macOS (arm64, x64), Linux glibc (x64, arm64), and Windows x64; elsewhere, build with cargo.

```bash
npm i -g napl-lang        # prebuilt binary via optionalDependencies
npx napl-lang init        # or run once, without installing
```

```bash
curl -fsSL https://raw.githubusercontent.com/conciv-dev/napl/main/install.sh | sh
```

Installs into `~/.local/bin`, or `$NAPL_INSTALL`.

```bash
cargo install --git https://github.com/conciv-dev/napl napl-cli
```

## Backends

The coding agent always runs via the `claude` CLI. The backend in `.napl/lock.json` selects how the derived IR and attribution completions are made:

| Backend | How | Needs |
| --- | --- | --- |
| `claude-cli` (default) | Shells out to Claude Code, reusing your login | `claude` on `PATH` |
| `anthropic-api` | Calls the Anthropic API directly | `ANTHROPIC_API_KEY` |

A `lock.json` without a `backend` field means `claude-cli`.
