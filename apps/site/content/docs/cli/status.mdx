---
title: napl status
description: Report clean / prompt-stale / DRIFT / unattributed per prompt. Non-zero exit on drift, so it works as a CI gate.
---

```text title="usage"
Usage: napl status

Options:
  -h, --help  Print help
```

| State | Meaning | Exit |
| --- | --- | --- |
| `clean` | Prompt unchanged since last gen, locked files intact | 0 |
| `prompt-stale` | Prompt changed; regen pending | 0 |
| `DRIFT` | A locked file was edited by hand | 1 |
| `unattributed` | Generated files exist but attribution failed | 1 |

```bash
napl status
```

Exit `1` makes it a direct CI gate; the `pre-commit` hook from `napl init` runs it and blocks the commit (escape hatch: `git commit --no-verify`).
